## 1. Essential Tools (Left Toolbar)

Activate tools instantly with single-key shortcuts.

| Tool Name | Shortcut | Action Description |
| :--- | :--- | :--- |
| **Move** | `V` | Select, move, and scale layers. |
| **Scale** | `K` | Resize elements proportionally (including font size/stroke width). |
| **Frame** | `F` or `A` | Draw a new artboard/frame coordinate space. |
| **Slice** | `S` | Draw a slice region for custom exporting. |
| **Rectangle** | `R` | Draw a rectangle. Hold `Shift` to lock square ratio. |
| **Line** | `L` | Draw a line. Hold `Shift` to snap angles to 45°. |
| **Arrow** | `Shift + L` | Draw a directional arrow vector line. |
| **Ellipse** | `O` | Draw a circle/ellipse. Hold `Shift` to lock aspect ratio. |
| **Polygon** | *(None)* | Draw triangles, stars, or multisided shapes. |
| **Pen** | `P` | Standard vector pen tool. |
| **Pencil** | `Shift + P` | Freehand drawing tool. |
| **Text** | `T` | Draw text boxes or type inline. |
| **Hand (Pan)** | `H` | Drag around the canvas. Alternatively, hold `Space` and drag. |
| **Comments** | `C` | Enter comment mode to place pin annotations. |

| Action | Shortcut |
| :--- | :--- |

| **Open Layers Panel** | `Option + 1` |
| **Open Assets Panel** | `Option + 2` |

| **Zoom to 100% (Actual Size)** | `Shift + 0` |
| **Zoom to Fit Selection** | `Shift + 2` |
| **Zoom to Fit Canvas Contents** | `Shift + 1` |
| **Zoom to Next Frame** | `N` |
| **Zoom to Previous Frame** | `Shift + N` |
| **Show/Hide Layout Grids** | `Ctrl + G` |
| **Show/Hide Rulers** | `Shift + R` |
| **Toggle Outline / Wireframe Mode** | `Cmd + Option + Y` |
| **Toggle Pixel Preview** | `Ctrl + P` |
| **Toggle Pixel Snap** | `Cmd + Shift + '` |

Align elements quickly relative to each other or their parent frame.

| Align Action | Shortcut | Layout Action | Shortcut |
| :--- | :--- | :--- | :--- |
| **Align Left** | `Option + A` | **Group Selection** | `Cmd + G` |
| **Align Right** | `Option + D` | **Ungroup Selection** | `Cmd + Shift + G` |
| **Align Top** | `Option + W` | **Frame Selection** | `Cmd + Option + G` |
| **Align Bottom** | `Option + S` | **Bring to Front** | `Cmd + Shift + ]` |
| **Align Horiz. Center** | `Option + H` | **Bring Forward** | `Cmd + ]` |
| **Align Vert. Center** | `Option + V` | **Send Backward** | `Cmd + [` |
| **Distribute Horiz. Spacing** | `Ctrl + Option + H` | **Send to Back** | `Cmd + Shift + [` |
| **Distribute Vert. Spacing** | `Ctrl + Option + V` | **Flatten Layers** | `Cmd + E` |
| **Tidy Up Grid** | `Ctrl + Option + Shift + T` | **Outline Stroke** | `Cmd + Shift + O` |

## 7. Vector Editing Mode

Shortcuts applicable when double-clicking a vector path or pressing `Enter` to enter Vector Node Edit mode.

| Action | Shortcut |
| :--- | :--- |
| **Pen Tool** | `P` |
| **Bend Vector Tool (Bezier curvature)** | `Cmd` *(hold while hovering path or node)* |
| **Paint Bucket Tool (Fill area)** | `B` |
| **Join Vector Paths** | `Cmd + J` |
| **Delete Selected Node** | `Delete` or `Backspace` |
| **Break Vector Path at Node** | `Cmd + Option + Click` node |
| **Toggle Curved vs. Straight Corner Node** | `Double Click` node or hold `Cmd` + Click |
| **Exit Vector Editing Mode** | `Enter` or click **Done** on top toolbar |
