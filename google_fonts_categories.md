## 1. Sans-Serif Fonts
Sans-Serif typefaces lack decorative strokes (serifs) at the ends of their letterforms. They represent modernism, clean geometry, and optimal digital readability.

| Font Family | Sub-Genre | Characteristics | Best Used For |
| :--- | :--- | :--- | :--- |
| **Roboto** | Neo-Grotesque | Neutral, friendly, dual-natured curves | General body text, Android UI |
| **Inter** | Neo-Grotesque | Extremely high legibility, optimized for screen UI | SaaS Dashboards, tiny UI labels |
| **Open Sans** | Humanist | Neutral, friendly, highly readable | Web articles, body copy, mobile |
| **Lato** | Humanist | Semi-rounded corners, warm, professional | Corporate websites, landing pages |
| **Montserrat** | Geometric | Wide capital letters, vintage poster vibe | Modern landing page headings |
| **Poppins** | Geometric | Circular forms, balanced weight spreads | App UI, friendly SaaS headers |
| **Plus Jakarta Sans**| Geometric | Clean, premium, Indonesian geometric | High-end tech landing pages |
| **Nunito** | Humanist | Rounded terminals, soft, friendly | E-learning, kids apps, callouts |
| **Source Sans 3** | Grotesque | Highly functional design-system standard | Extensive administrative data sheets |
| **Ubuntu** | Neo-Grotesque | Distinctive elliptical shapes, technical | Linux environments, tech portals |
| **Oswald** | Grotesque | Condensed, tall uppercase profiles | Compact headings, sidebars |
| **PT Sans** | Humanist | Based on Russian sans serif types | Multilingual sites, localized copy |
| **DM Sans** | Geometric | Low-contrast, clean, geometric | Mobile-first headers, minimal cards |
| **Work Sans** | Grotesque | Wide proportions, optimized for screens | Bold editorial titles, clean copy |
| **Quicksand** | Geometric | Rounded corners, thin circular frames | Casual UI, friendly, child-focused |

| **Noto Serif** | Transitional | Universal language script coverage | Multi-language localized articles |

## 3. Display Fonts
Display typefaces are designed for high-impact viewing at large sizes (usually 32px or larger). They prioritize personality, visual texture, and decorative elements over tiny text legibility.

| Font Family | Style / Vibe | Visual Highlights | Best Used For |
| :--- | :--- | :--- | :--- |
| **Syne** | Avant-Garde | Extra-wide bold weights, eccentric curves | Creative portfolios, web agencies |
| **Outfit** | Geometric Display | circular outlines, clean geometric | Modern minimalist SaaS headings |
| **Bebas Neue** | Condensed Bold | Tall, narrow uppercase letters | YouTube thumbnails, movie titles |
| **Abril Fatface** | Didone Display | Ultra-bold, high-contrast curves | Bold editorial titles, magazines |
| **Lobster** | Script Display | Playful, hand-drawn vector script | Casual signage, food packaging |
| **Cinzel Decorative**| Ornate Roman | Classical letters with extended swashes | Fantasy branding, wedding headers |
| **Pacifico** | Retro Script | Smooth, brush script, Hawaiian vibe | Fun logos, ice cream parlors |
| **Anton** | Heavy Sans | Thick, impactful sans-serif bars | Sports marketing, massive banners |
| **Alumni Sans** | Ultra-Condensed | Tall, needle-thin vertical columns | Tech specifications, side metrics |
| **Righteous** | Art Deco | Rounded geometry, unique lowercase designs | Synthwave titles, gaming platforms |
| **Cabinet Grotesk** | Neo-Grotesque | Angled terminals, extreme contrast | Creative branding, editorial headers |
| **Clash Display** | Geometric Display | High visual weights, sharp corners | Design magazines, typography show |

## 5. Monospace Fonts
Monospace typefaces have characters that all occupy the exact same horizontal width. They are essential for alignment-sensitive text.

| Font Family | Coding Ligatures | Style Details | Best Used For |
| :--- | :--- | :--- | :--- |
| **Fira Code** | Yes | Clean, sharp, built-in operator symbols | Code blocks, IDE workspaces |
| **JetBrains Mono**| Yes | Tall x-height, comfortable coding metrics | Coding fonts, documentation |
| **Source Code Pro**| No | Highly legible, clean interface design | Terminal output windows |
| **Inconsolata** | No | Narrow, elegant vertical columns | Command prompts, log outputs |
| **DM Mono** | No | Wide, geometric, tech aesthetic | Pricing packages, dashboards |
| **Roboto Mono** | No | Neutral, matches Roboto system layout | Tabular data sheets, input logs |
| **IBM Plex Mono** | No | Industrial, mechanical styling | Technical charts, engineering logs |
| **Space Mono** | No | Retro-futuristic, wide proportions | Tech startups, dashboard grids |
| **Cutive Mono** | No | Typewriter style font | Literary blogs, artistic wireframes |

## 6. Type Classification Reference Guide

Use this flowchart to determine which category of font is appropriate for your layout needs:

```
                  Is the text block for reading or decoration?
                                    │
                  ┌─────────────────┴─────────────────┐
            [ Reading ]                         [ Decoration ]
                  │                                   │
      Is it code or paragraph text?            Which aesthetic vibe?
        ┌─────────┴─────────┐                   ┌─────┼─────────┐
    [ Code ]           [ Paragraph ]         [ Bold ] [Personal][Classic]
        │                   │                   │        │         │
   (Monospace)      Do you want modern          │   (Handwriting)  │
                    or traditional?             │                  │
               ┌────────────┴────────────┐  (Display)          (Cinzel/Playfair)
           [ Modern ]            [Traditional]
               │                       │
          (Sans-Serif)              (Serif)
```
