## 1. Affinity Designer Persona Tools

Quickly switch tools in Affinity Designer’s vector environment.

| Tool Name | Shortcut | Action Description |
| :--- | :--- | :--- |
| **Move Tool** | `V` | Select, transform, and move layers. |
| **Node Tool** | `A` | Edit vector curve points (anchor points and handles). |
| **Corner Tool** | `C` | Round or chamfer vector corner points. |
| **Pen Tool** | `P` | Draw precise vector paths with Bezier curves. |
| **Pencil Tool** | `N` | Draw organic freehand vector paths. |
| **Vector Brush Tool** | `B` | Paint vector stroke paths using texture brushes. |
| **Fill Tool** | `G` | Apply and edit solid, linear, radial, or conical gradients. |
| **Transparency Tool** | `Y` | Apply transparency gradients to elements. |
| **Shape Tools Cluster** | `M` | Cycle through Rectangle, Rounded Rectangle, and Ellipse. |
| **Vector Crop Tool** | `K` | Crop individual elements or layers non-destructively. |
| **Artistic Text / Frame Text** | `T` | Cycle between creative and block paragraph text tools. |
| **Color Picker Tool** | `I` | Sample colors directly from the document canvas. |
| **View Tool (Hand)** | `H` | Pan across the canvas (or hold `Space` + Drag). |
| **Zoom Tool** | `Z` | Drag to zoom into sections of the artboard. |

## 5. Layer Management & Arrangement

Order elements, group structures, and control blending within the Layer tree.

| Arrange Action | Shortcut | Layer Action | Shortcut |
| :--- | :--- | :--- | :--- |
| **Move to Front** | `Ctrl + Shift + ]` | **Group Selected** | `Ctrl + G` |
| **Move Forward** | `Ctrl + ]` | **Ungroup Selected** | `Ctrl + Shift + G` |
| **Move Backward** | `Ctrl + [` | **Create New Layer** | `Ctrl + Alt + N` |
| **Move to Back** | `Ctrl + Shift + [` | **Hide Selected Layer** | `Ctrl + H` |
| **Align Left** | `Ctrl + Alt + Left Arrow` | **Rasterize Layer** | *(Via Right-Click Menu)* |
| **Align Right** | `Ctrl + Alt + Right Arrow` | **Toggle Solo Layer (Isolate)** | `Alt + Left Click` layer eye |
| **Align Top** | `Ctrl + Alt + Up Arrow` | **Select Multiple Layers** | `Ctrl + Left Click` in Layers panel |
| **Align Bottom** | `Ctrl + Alt + Down Arrow` | **Opacity Step Keys** | `1` to `9` (with layer selected) |

## 7. Node & Vector Path Editing

Shortcuts active when manipulating points using the **Node Tool (`A`)**.

| Action | Shortcut |
| :--- | :--- |
| **Constrain Anchor Position to 45°** | Hold `Shift` while dragging point |
| **Split Curve at Node** | Select Node → Click **Break Curve** on Context Toolbar |
| **Join Vector Paths** | Select overlapping end nodes → `Ctrl + Shift + J` |
| **Convert Node to Sharp Corner** | Hold `Ctrl + Alt` + Click Node |
| **Convert Node to Smooth Curve** | Hold `Ctrl` + Click Node |
| **Temporarily Toggle to Move Tool** | Hold `Ctrl` |
| **Select All Nodes on Path** | `Ctrl + A` *(with Node Tool active on path)* |
| **Nudge Anchor Point (1px)** | `Arrow Keys` |
| **Nudge Anchor Point (10px)** | `Shift + Arrow Keys` |
